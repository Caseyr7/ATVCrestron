# Pinned dependency set for pyatv 0.15.1 on Crestron CP4 (Python 3.8.2, armv7l).
#
# 0.15.1 is the LAST pyatv release that supports Python 3.8 - all later versions
# require Python >=3.9 which the CP4 firmware does not provide.
#
# Version pins are chosen to:
#   - satisfy pyatv 0.15.1's requires_dist (aiohttp<5,>=3.8.3 etc.)
#   - stay on the pure-Python branch of packages that went Cython/Rust in newer majors
#     (pydantic<2 avoids Rust, protobuf<5 stays pure-Python)
#   - prefer cp38-abi3 wheels where they exist (cryptography 46.x)
#
# miniaudio is intentionally omitted - pyatv imports it but the installer stubs
# it out since we do not stream audio on the CP4.
